# タイトル
病棟共有空間における空間的文脈を考慮した危険性評価システム

# 投稿先
IEEE Access または IEEE Transactions on Systems, Man, and Cybernetics

# 1. Introduction
- 病棟共有空間における見守り業務負荷が増大している
- 見守り業務の効率化に向け、見守り対象者のリスク評価システムの需要が高まっている
- 患者の危険性は本人の属性や行動（以後、主体的文脈）だけでなく、周囲の物体や人員配置（以後、空間的文脈）を含めた、多特徴量を総合的に評価して決まる
- 評価における重み付けや特徴量と危険度の関係性に関する知識（以後、現場知識）は、スタッフにより異なる
- 本研究では、見守り業務の負荷軽減のための見守り支援室エムの実現に向け、患者の空間的文脈を考慮し、かつスタッフ毎に異なる現場知識を反映可能な、危険性評価手法を提案する

# 3. Proposed Method
# 3.1 Problem Setting
- 実現したいこと：現場知識（危険度評価に際する重み付けや特徴量と危険度の関係性）に基づき、空間的文脈を考慮して患者の危険度評価を行い、スタッフに伝達する仕組み
- 上記実現に際する課題
  - 課題1. 空間的文脈を含む、患者とその周囲の多くの特徴量から危険度を導くこと
    - 従来研究では、患者の属性や行動などの主体的文脈を中心に評価されており、特徴量が限定的であったため、課題1が問題になることは少なく、扱われてこなかった
  - 課題2. 危険度を導く際の、特徴量間の重み付けや特徴量と危険度の関係性（現場知識）を、スタッフ一人一人の特性に基づいて調整できること
    - 従来研究では、Alarm fatigueの問題に対処するため、危険度評価の閾値を調整する研究が行われているが、スタッフによって必要とする通知が異なることや、スタッフ毎に通知を調整しやすくする仕組みは十分に検討されていない
  - 課題3. 危険度の根拠を説明可能であること
    - ナースコールを鳴らす際の閾値を調整する研究は行われているが、危険度の根拠を説明可能にする研究は十分に検討されていない

# 3.2 Approach
- 多特徴量から危険度を導くという問題構造に対しては、機械学習のアプローチが考えられる。
- しかし、病棟見守りやそれに際する現場知識のデータセットは少ない他、課題2を解決することと、課題3を担保する観点から、学習ベースではなく、意思決定法によるアプローチを採用する。
- 意思決定法に基づく際、課題1に対応するためには、多くの特徴量を扱う必要があるが、1層での考慮は困難である。（例：「年齢」と「重症度」のどちらが危険性により寄与するかは比較的答えやすいが、「重症度」と「周囲に手すりがあるか」のどちらが危険性に寄与するかは、両特徴量の性質が異なるため答えづらい）
- 従って、カテゴリごとに特徴量を順次集約する階層構造とした。
- 階層構造のうち、下層では、特徴量間で危険性に寄与する優劣が比較的明確に答えやすい。一方、上層では、特徴量間の相乗効果といった組み合わせに関する知識を反映する必要があることが分かった。そこで、下層はAHP、上層はFuzzy推論を用いることとした。

# 4. Experiments
# 4.1 Objective
- 空間的文脈を考慮することで、より実際に危険性が高い患者を高い危険度で評価できることの確認
- 異なる現場知識を持つ実際のスタッフへのアンケート結果を用いて評価した際、知識に応じ異なる評価が出せることの確認
# 4.2 Experimental Setup
- Simulation
  - 空間的文脈の異なる3人の患者が相次いで危険な行動（立ち上がる/顔を触る）をとるシナリオを模擬的に生成し、データを評価した際に、3人の危険度が異なることの確認
- Experiment
  - 実データを、2人のスタッフのアンケート結果を用いて評価した際、スタッフ毎に異なる危険度が出ることの確認
# 4.3 Results
- Simulation
  - 空間的文脈を考慮することで、見守られていない患者に高い危険度が示されるなど、より実際に危険性が高い患者を高い危険度で評価できることが確認された
- Experiment
  - スタッフ毎に異なる危険度が出ることが確認された。空間的文脈に関係なく通知を望むスタッフには、立ち上がる・顔を触るといった危険行動に対して高い危険度が示される一方、空間的文脈を考慮して通知を望むスタッフには、見守られていない患者の危険な行動に対して高い危険度が示された
# 5. Conclusion
# 5.1 Summary
- 空間的文脈を含む他特徴量から、スタッフのアンケート結果から導いたAHPの重み・Fuzzyルールを用いて危険度を算出することにより、スタッフにとって真にちゅいすべき患者を検出し、通知できるシステムを提案した。
- Alarm fatigueの防止と事故防止の両立に向けた１手法として期待でき、見守り業務負荷軽減への貢献が期待できる。
