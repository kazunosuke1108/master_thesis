# タイトル

病棟共有空間における空間的文脈とスタッフ知識を考慮した危険度評価システム

# 投稿先

IEEE Access または IEEE Transactions on Systems, Man, and Cybernetics

# 1. Introduction

* 病棟共有空間における見守り業務では、患者の状態確認や危険行動への対応など、スタッフに大きな負荷が生じている。
* 見守り業務の効率化に向けて、見守り対象者の状態を評価し、必要な通知を行うリスク評価システムの需要が高まっている。
* 病棟共有空間における患者の危険度は、患者本人の属性や行動（以後、患者文脈）だけでなく、周囲の物体、スタッフ配置、他者との位置関係などの空間的文脈にも影響を受ける。
* 例えば、同じ立ち上がり動作であっても、近くにスタッフや手すりがある場合と、周囲に誰もおらず支援可能な物体もない場合では、危険度は異なると考えられる。
* さらに、どのような状況を危険と判断するか、またどの患者を優先的に通知すべきかに関する判断基準は、スタッフの経験や担当業務によって異なる可能性がある。
* そのため、病棟共有空間における見守り支援では、患者文脈と空間的文脈を統合した危険度評価に加えて、スタッフごとの現場知識を反映可能であり、かつ危険度の根拠を説明可能な評価手法が必要である。
* 本研究では、病棟共有空間における見守り支援システムの実現に向けて、空間的文脈とスタッフごとの現場知識を考慮した危険度評価手法を提案する。
* 本研究の貢献は以下の3点である。

  * 患者文脈と空間的文脈を階層的に統合する危険度評価モデルを構築したこと。
  * スタッフごとの現場知識をAHPの重みおよびFuzzyルールとして反映可能な個別化手法を提案したこと。
  * 算出された危険度に加えて、その根拠を提示する通知機構を設計したこと。

# 2. Related Work

* 患者見守りシステム

  * 既存の患者見守りシステムでは、バイタルサイン、姿勢、行動、離床、転倒など、患者本人の状態や行動に基づく検知・通知が多く検討されてきた。
  * 一方で、病棟共有空間では、患者本人の状態だけでなく、周囲の物体、スタッフの位置、他者との関係といった空間的文脈が危険度に影響する。
  * しかし、空間的文脈を含む多特徴量を統合し、患者の危険度として評価する研究は十分に検討されていない。
* 通知調整・Alarm fatigue対策

  * 医療現場では、過剰な通知や不要なアラームによってスタッフの負荷が増大し、重要な通知への対応が遅れるAlarm fatigueの問題が指摘されている。
  * そのため、通知閾値の調整やアラーム条件の最適化に関する研究が行われている。
  * しかし、スタッフごとに異なる危険判断の傾向や通知ニーズを、危険度評価モデルの重みやルールとして反映する仕組みは十分に検討されていない。
* 意思決定支援法

  * OR・システム工学の分野では、多特徴量に基づいて評価や意思決定を行うための手法として、AHPやFuzzy推論などが提案されている。
  * AHPは、特徴量間の重要度を階層的に整理し、重み付けを行う手法として有用である。
  * Fuzzy推論は、複数の条件の組み合わせに基づいて、曖昧な判断知識をルールとして表現する手法として有用である。
  * しかし、これらの意思決定支援法を、病棟共有空間における患者危険度評価へ適用し、空間的文脈とスタッフごとの現場知識を同時に扱う研究は十分に行われていない。
* 以上より、本研究では、病棟共有空間における見守り支援を対象として、空間的文脈、スタッフごとの現場知識、説明可能な通知を統合した危険度評価手法を提案する。

# 3. Proposed Method

## 3.1 Problem Setting

* 本研究で実現したいことは、スタッフの現場知識に基づき、患者文脈と空間的文脈を考慮して患者の危険度を評価し、その結果と根拠をスタッフに提示する仕組みである。
* ここで現場知識とは、危険度評価における特徴量間の重み付けや、特徴量と危険度の関係性に関するスタッフの判断知識を指す。
* 上記を実現するための課題を以下に示す。

### 課題1：空間的文脈を含む多特徴量から危険度を導くこと

* 病棟共有空間では、患者の属性や行動だけでなく、周囲の物体、スタッフ配置、他者との位置関係などが危険度に影響する。
* そのため、患者本人に関する特徴量と周囲環境に関する特徴量を統合して評価する必要がある。
* 従来研究では、患者の属性や行動などの患者文脈を中心に評価するものが多く、空間的文脈を含む多特徴量を危険度評価に統合する仕組みは十分に扱われていない。

### 課題2：スタッフごとの現場知識を危険度評価に反映すること

* 危険度を導く際には、特徴量間の重み付けや、特徴量の組み合わせと危険度の関係を設定する必要がある。
* しかし、どの状況を危険と判断するか、どの患者を優先的に通知すべきかは、スタッフの経験や担当業務によって異なる可能性がある。
* 従来研究では、Alarm fatigueへの対策として通知閾値を調整する研究は行われているが、スタッフごとの判断知識を危険度評価モデルの重みやルールとして反映する仕組みは十分に検討されていない。

### 課題3：危険度の根拠を説明可能にすること

* 見守り支援システムが通知を行う際には、単に危険度を提示するだけでなく、なぜその患者が危険と判断されたのかをスタッフが理解できることが望ましい。
* 危険度の根拠が示されることで、スタッフは通知の妥当性を判断しやすくなり、対応の優先順位付けにも活用しやすくなる。
* 従来研究では、通知条件や閾値の調整は検討されているが、危険度の根拠を空間的文脈と関連付けて提示する仕組みは十分に検討されていない。

## 3.2 Approach

* 多特徴量から危険度を導く問N題に対しては、機械学習に基づくアプローチが考えられる。
* しかし、病棟共有空間における見守り場面では、空間的文脈を含む十分な教師データを収集することが容易ではない。
* また、本研究では、スタッフごとの現場知識を反映できること、および危険度の根拠を説明できることを重視する。
* そのため、本研究では学習ベースの手法ではなく、意思決定支援法に基づく危険度評価手法を採用する。
* 空間的文脈を含む多特徴量を扱う場合、すべての特徴量を同一層で直接比較することは困難である。

  * 例えば、「年齢」と「重症度」のどちらが危険度に寄与するかは比較的判断しやすい。
  * 一方で、「重症度」と「周囲に手すりがあるか」のどちらが危険度に寄与するかは、特徴量の性質が異なるため直接比較しづらい。
* そこで本研究では、特徴量をカテゴリごとに整理し、段階的に危険度へ集約する階層構造を採用する。
* 下層では、同じカテゴリ内の特徴量を比較し、危険度への寄与度を重み付けするためにAHPを用いる。
* 上層では、患者文脈と空間的文脈の組み合わせによって危険度が変化することを表現するために、Fuzzy推論を用いる。
* AHPのみを用いる場合、特徴量の階層的な重み付けは可能であるが、「危険行動をとっているが近くにスタッフがいる場合」と「危険行動をとっており周囲に支援者がいない場合」のような条件の組み合わせを表現しづらい。
* 一方で、Fuzzy推論のみを用いてすべての特徴量を直接扱う場合、ルール数が増加し、スタッフが知識を設定・調整する負担が大きくなる。
* そこで本研究では、下層でAHPにより特徴量を整理し、上層でFuzzy推論により文脈の組み合わせを扱う階層型AHP-Fuzzy評価手法を提案する。

## 3.3 Implementation

### 3.3.1 Feature Selection and Extraction

* 患者文脈および空間的文脈に関する特徴量を定義する。
* 患者文脈には、患者の属性、状態、行動などに関する特徴量を含める。
* 空間的文脈には、周囲の物体、スタッフ配置、他者との位置関係などに関する特徴量を含める。
* 各特徴量は、センサ情報、位置情報、事前に設定された環境情報などから抽出する。

### 3.3.2 Risk Calculation

* 抽出された特徴量をカテゴリごとに整理し、下層ではAHPにより各カテゴリ内の危険度を算出する。
* AHPの重みは、スタッフへのアンケート結果に基づいて設定する。
* 上層では、下層で算出されたカテゴリごとの評価値を入力として、Fuzzy推論により最終的な危険度を算出する。
* Fuzzyルールは、スタッフの現場知識を反映して設定する。
* これにより、同一の患者状況であっても、スタッフごとの判断傾向に応じた危険度評価が可能となる。

### 3.3.3 Notification Generation

* 算出された危険度に基づき、通知対象となる患者を決定する。
* 通知では、危険度の値だけでなく、危険度に寄与した主な要因を提示する。
* 例えば、患者の危険行動、周囲にスタッフがいないこと、支援可能な物体が近くにないことなどを通知理由として提示する。
* これにより、スタッフは通知の根拠を理解し、対応の優先順位を判断しやすくなることが期待される。

# 4. Experiments

## 4.1 Objective

* 本実験の目的は、提案手法が空間的文脈およびスタッフごとの現場知識の違いを危険度評価に反映できることを確認することである。
* 具体的には、以下の2点を確認する。

  * 空間的文脈の違いに応じて、同様の危険行動をとる患者であっても異なる危険度が算出されること。
  * スタッフごとの現場知識の違いに応じて、同一の状況に対して異なる危険度が算出されること。
* なお、本実験は提案手法の基礎的な挙動確認を目的とするものであり、業務負荷軽減や事故防止の効果を直接実証するものではない。

## 4.2 Experimental Setup

### Simulation

* 空間的文脈の異なる3人の患者が、相次いで危険行動をとるシナリオを模擬的に生成する。
* 危険行動として、立ち上がりや顔を触る動作を設定する。
* 各患者について、周囲のスタッフ配置や物体配置が異なる条件を設定する。
* 提案手法を用いて各患者の危険度を算出し、空間的文脈の違いが危険度に反映されるかを確認する。

### Experiment

* 実データを用いて、スタッフへのアンケート結果に基づく危険度評価を行う。
* 異なる判断傾向を持つ2名のスタッフのアンケート結果を用いて、AHPの重みおよびFuzzyルールを設定する。
* 同一の患者状況に対して、スタッフごとの現場知識の違いが危険度に反映されるかを確認する。

## 4.3 Results

### Simulation

* 空間的文脈を考慮することで、同様の危険行動をとる患者であっても、周囲の状況に応じて異なる危険度が算出された。
* 例えば、周囲にスタッフがいない患者や、支援可能な物体が近くにない患者に対して、相対的に高い危険度が示された。
* この結果から、提案手法が患者本人の行動だけでなく、空間的文脈の違いを危険度評価に反映できることを確認した。

### Experiment

* スタッフごとのアンケート結果に基づいて評価した結果、同一の患者状況に対しても、スタッフごとに異なる危険度が算出された。
* 空間的文脈に関係なく危険行動への通知を望む傾向を持つスタッフでは、立ち上がりや顔を触る動作に対して高い危険度が示された。
* 一方で、空間的文脈を考慮して通知を望む傾向を持つスタッフでは、危険行動そのものだけでなく、周囲にスタッフがいないことなどの条件を踏まえて高い危険度が示された。
* この結果から、提案手法がスタッフごとの現場知識の違いを危険度評価に反映できることを確認した。

# 5. Conclusion

## 5.1 Summary

* 本研究では、病棟共有空間における見守り支援に向けて、空間的文脈とスタッフごとの現場知識を考慮した危険度評価手法を提案した。
* 提案手法では、患者文脈と空間的文脈に関する多特徴量を階層的に整理し、下層ではAHPにより特徴量の重み付けを行い、上層ではFuzzy推論により文脈の組み合わせを考慮した危険度を算出する。
* また、スタッフへのアンケート結果に基づいてAHPの重みおよびFuzzyルールを設定することで、スタッフごとの現場知識を危険度評価に反映可能とした。
* 基礎評価により、提案手法が空間的文脈の違い、およびスタッフごとの現場知識の違いを危険度に反映できることを確認した。
* さらに、危険度に寄与した要因を通知理由として提示することで、スタッフが通知の根拠を理解しやすい見守り支援につながる可能性を示した。
* 本手法は、病棟共有空間における見守り業務負荷軽減やAlarm fatigue対策に向けた基礎的手法として活用できる可能性がある。

## 5.2 Limitations

* 本研究の評価は、提案手法の基礎的な挙動確認を目的としたものであり、危険度評価の正解データに対する精度検証は行っていない。
* また、業務負荷軽減、Alarm fatigueの低減、事故防止効果については、実際の医療現場における長期的な検証が必要である。
* スタッフごとの現場知識を反映する場合、経験不足や誤った判断傾向まで反映される可能性があるため、安全性を担保するための共通ルールや通知下限値の設計が必要である。

## 5.3 Future Work

* より多くのスタッフおよび患者状況を対象とした評価を行い、提案手法の妥当性を検証する。
* 複数スタッフにシナリオの危険度や通知優先順位を評価してもらい、提案手法の出力との一致度を検証する。
* 空間的文脈を考慮しない手法、個別化を行わない手法、AHPのみを用いる手法などとの比較により、提案手法の有効性を検証する。
* 通知理由の理解しやすさ、納得度、有用性について、スタッフへのアンケート評価を行う。
* スタッフごとに通知内容が異なることによる情報共有上の課題や、通知漏れを防ぐための安全設計について検討する。
